# Projeto Podcast Gerado por I.A.s


  > ℹ️ **NOTE:** Este é o repositório desenvolvido durante o curso [Formação ChatGPT for Devs](https://web.dio.me/track/formacao-chatgpt-devs) na plataforma da [DIO](https://dio.me).

Projeto com o objetivo de gerar um podcast utilizando ferramentas de IA através de prompts mais trabalhado.

Utilizer uma esteira de prompts para gerar cada etapa do processo criativo.

## 💻 Tecnologias utilizadas no projeto

- [NotebookLM](https://notebooklm.google.com/) 
- [Claude](https://claude.ai/)
- [Gemini](https://gemini.google.com/)

## ✨ Como foi feito ?

- Roteiro gerado via Claude a partir de conteúdo do curso na aula https://github.com/rsl50/prompts-for-article-generate-by-ia
- Audio gerado pelo NotebookLM
- Gemini Para gerar capas
